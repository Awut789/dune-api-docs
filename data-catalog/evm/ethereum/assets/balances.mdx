Our daily balances tables contain all token balances for all EVM addresses. This includes:

- **native (ETH, MATIC, etc.) balances**
- **ERC-20 token balances**
- **ERC-721 token balances**
- **ERC-1155 token balances**

## Usage

The balances tables are a great starting point for any analysis that requires token balances. For example, you can use the balances tables to:
- **Analyze the distribution of token holdings**
- **Identify the top token holders**
- **Monitor the movement of tokens**
- **Analyze a wallet's token holdings**
- **Inspect the holdings of a multi-sig wallet**

and much more.

### Coverage

These tables currently cover the following networks:

- Arbitrum
- Base 
- Ethereum Mainnet
- Optimism

We are working on adding support for all EVM networks.
